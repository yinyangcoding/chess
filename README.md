# Chess

Chess is a basic chess game written in C++. In the future it will be expanded to support a GUI and AI

## Executing

You can use make to execute the code

**MacOS and GNU/Linux:**
```bash
make run
```

**WindowsOS:**
```bash
make runWin
```

## Usage

**Enter the to and from coordinates as seen below**

    A B C D E F G H
